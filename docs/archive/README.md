# Archive

These are historical, one-off documents from past development sessions. They are
kept for reference but are **no longer maintained** — dates, version numbers, and
file references inside them describe the project as it was at the time of writing,
not as it is today.

For current information, start with the root [README](../../README.md) and
[CHANGELOG](../../CHANGELOG.md).

| File | What it was |
|------|-------------|
| `PROJECT_STATUS.md` | Status snapshot from January 2026 (project already 100% complete at that point). References several files that have since been deleted. |
| `BUGFIX_PLACEHOLDER_TEXT.md` | Detailed handoff notes for the v3.3.8 placeholder-text bug fix (February 2026). The fix itself is live; the summary lives in `CHANGELOG.md`. |
| `BUG_FIX_DEFAULT_TEXT.md` | Shorter, earlier write-up of the same placeholder-text bug fix. |
| `DEPLOYMENT_SUMMARY_ORIGIN_TRACKING.md` | Deployment record for the QR-code origin-tracking feature (January 2026). Mentions an `ORIGIN_TRACKING.md` that no longer exists in the repo. |
| `TRANSLATION_HEBREW_SOURCE.md` | Working document: all Hebrew UI strings extracted for translation. |
| `TRANSLATION_ENGLISH_CLAUDE.md` | Working document: the English translation of those strings. |
| `TRANSLATION_MASTER.md` | Working document: translation guidelines and the master Hebrew/English table. Useful reference if translation work continues — the live strings are in `src/i18n/locales/`. |
